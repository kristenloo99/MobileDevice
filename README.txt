Mobile Device Keyboard

Download the zip file "MobileDeviceKeyboard.zip" and unzip the file to a directory.
From a command line, run the program by going into that directory and typing "java -classpath <directory path> Keyboard"

The prompt will ask you to "Enter 1 to train, 2 to input, 3 to display, 0 to exit: "
When entering 1, you train the keyboard by entering in a passage.
When entering 2, you input a fragment to the keybaord, which will then give you the autocomplete suggestions and number of occurences for that fragment.
When entering 3, it will display all previously trained sentences.
When entering 0, the program will end.
